This plugin allows you to add ajax driven drop down cart for your woocommerce store using shortcode [nc_ajax_cart]

== Description ==
Woocommerce default features dont allow you to manage cart via ajax.But dont worry NC Ajax Cart for woocommerce  will simply add that feature to your woocommerce store via shortcode [nc_ajax_cart] anywhere in your theme. This Plugin enable ajax cart features for simple product, variable product , grouped product.

== Installation Guide ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the setting link from plugin page.
